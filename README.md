 # 幻想乡外的同学们
《幻想乡外的同学们》是一个由FandaP开发，使用Ren'Py引擎制作的视觉小说。  
  
Ren'Py是一款免费且跨平台的数字叙事引擎。它让你可以轻松组合文字、图片和声音，制作视觉小说和生活模拟游戏。Ren'Py官方网站：https://renpy.org/  
Ren'Py的Github项目页：https://github.com/renpy/renpy/  
## 我可以为这个项目做什么？
对于初中车万众学习交流群的成员，你可以给这个仓库提交issues，但是issue中需要包含你在初中车万众学习交流群的昵称。  
对于其他人，你可以查看代码，并将其部分或全部使用在你的项目中，但不能提交issues。

# *StudentsOutsideGensokyo*
*Students Outside Gensokyo* is a visual novel which is developed by FandaP and powered by Ren'Py.  
  
Ren'Py is a free and cross platform engine for digital storytelling. It makes it easy to combine words, images, and sounds to create visual novels and life simulation games.  
Ren'Py 's official website: https://renpy.org/  
Ren'Py 's github project page:https://github.com/renpy/renpy/  

## What can I do for this repository?
For TJHG members, you can submit issues to this repository, but you need to include your name in the issue.  
For other people, you can check the code in the repository and use the code in your own project, but you can't submit issues.
